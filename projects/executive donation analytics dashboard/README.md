# Executive Donation Analytics Dashboard

## Overview

Designed and developed a multi-page Power BI dashboard for a nonprofit organization to help leadership monitor donations, member engagement, fundraising performance, and regional trends.

---

## Business Problem

The nonprofit organization needed dashboards that could answer questions such as:

- Which regions contribute the highest donations?
- Which members donate the most?
- How do donations change over time?
- Which insights are most useful for different managers?

---

## My Role

I completed the entire analytics workflow:

- Cleaned and transformed raw datasets using Power Query
- Combined multiple datasets into a single data model
- Built relationships between tables
- Created interactive Power BI dashboards
- Designed dashboards for multiple managerial roles
- Generated business insights and recommendations
